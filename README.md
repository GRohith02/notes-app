# Notes App

A full-stack notes application with JWT-based authentication.

## Tech Stack

**Backend:** ASP.NET Core 8, Entity Framework Core, SQL Server, JWT Authentication, BCrypt

**Frontend:** React, React Router

## Features

- User registration and login (JWT-based authentication, passwords hashed with BCrypt)
- Create, view, and delete personal notes
- Notes are private to each authenticated user
- Protected routes (dashboard inaccessible without login)
